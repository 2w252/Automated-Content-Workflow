# Automated Content Workflow

Это веб-приложение для автоматизации создания и публикации контента из новостных лент (RSS) в Telegram-каналы с использованием искусственного интеллекта.

## 🌟 Возможности
*   **Импорт новостей (RSS):** Парсинг RSS-лент популярных ресурсов прямо из браузера через CORS прокси.
*   **ИИ-Рерайтер (Gemini API):** Генерация структурированных вовлекающих постов по тексту новости с добавлением смайликов и подходящих хэштегов с помощью модели Gemini Flash.
*   **Умный фоллбэк:** Возможность локальной генерации постов по встроенным шаблонам, если API-ключ Gemini отсутствует.
*   **Публикация в Telegram:** Мгновенный постинг готовых публикаций в ваш публичный или приватный Telegram-канал по клику.
*   **Терминал логов:** Логирование в реальном времени всех событий (парсинг ленты, генерация контента, отправка постов в Telegram) прямо на экране.

## 🛠️ Запуск проекта
Проект написан на чистом HTML5/CSS3/JavaScript и готов к использованию без сборки. 

Просто откройте файл `index.html` в любом браузере!
